# Coke-Sales-Analysis

This project presents an analysis of a Coke-Sales-Analysis Power BI dashboard, focusing on operational, geographical, maintenance, and financial performance. It provides a business-oriented interpretation of key metrics, visualizations, and KPIs, along with actionable insights to support data-driven decision-making. The analysis demonstrates how interactive dashboards can help stakeholders monitor fleet performance, optimize transportation operations, control costs, improve profitability, and identify trends through dynamic filtering and visual storytelling.
